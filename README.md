Do you hate Vernier Logger Pro? Is it crashing too much? Do you hate the User Interface? Well have I got the software for you.

Lumberjack Pro is a (much more stable) tool that allows users to track the motion of an object by placing pins in a video. A lot like Logger Pro. It just has additional Quality Of Life features and doesn't crash as much.

Features will be added on request! Just submit an Issue on the GitHub page (or ask me in-person if you know me)!

On Windows, you can just download one of the .zip files from the Releases tab, extract it, and run Lumberjack Pro.exe. **You need to have the other files in the folder for it to run.**

On Mac, things get much more complicated. Follow this guide if you need more help: https://docs.godotengine.org/en/latest/tutorials/export/running_on_macos.html

---

This software uses the Godot Game Engine alongside the GDE GoZen library by Voylin, protected under the LGPL-2.1 license, which is included inside of the addon's folder. You can find the source code for Voylin's library here: https://github.com/VoylinsGamedevJourney/gde_gozen
