Do you hate Vernier Logger Pro? Is it crashing too much? Do you hate the User Interface? Well have I got the software for you.

Lumberjack Pro is a (much more stable) tool that allows users to track the motion of an object by placing pins in a video. A lot like Logger Pro. It just has a functional eraser tool and doesn't crash as much.

Features will be added on request! Just submit an Issue on the GitHub page (or ask me in-person if you know me)!

---

This software uses the GDE GoZen library by Voylin.
